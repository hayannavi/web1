HTML이란 무엇인가?

나는 프론트엔드를 개발하는 웹개발자를 꿈꾸는 장태영이다.
그리고 나는 개인적으로 영화 리뷰 사이트를 만들고 싶다.
Hypertext Markup Language (HTML)

2019/02/13 html web 1일차

TAG = 열린 tag와 닫힌 tag로 사이의 코드에 대한 설명
strong = 진하게
u(underline) = 밑줄

h1 ~ h6 (headings) = 제목을 나타내는 태그

2019/02/14 html web 2일차

p과 br
정보로써 더 가치있는 html이 되는 것이 중요하다.
p는 코드에서 단락을 표현한다는 의미에서 더 가치를 가진다.
br은 단순한 줄바꿈 태그이다.

'잠깐! CSS'
style = 태그와 태그 사이에 여백을 나타내는 코드
ex: style = "margin-top:40px"

HTML의 태그의 용도를 알고 정확히 사용하는 것
비즈니스적이나 휴머니즘적으로 아주 중요하다.

tag의 심화 -> '속성'(Attribute)
태그의 이름만으로는 정보가 부족할 때가 있다.
속성의 위치는 상관없다.
img src = ""를 입력하므로써 추가적인 정보 제공
width = 100%를 적용하면 웹페이지 크기에 따라 자동으로 크기 변경

parent - child
부모 자식 관계처럼 사이가 아주 좋은 태그들이 있다.
html에서는 목차에 사용하라고 만든 태그가 있다.
li 태그는 부모 태그인 ul(Unordered List)을 꼭 갖고 있어야 한다. 반대도 마찬가지
ol(Ordered List)은 숫자가 자동으로 넘버링된다. ol도 li과 함께 다닌다.

문서의 구조를 표현하는 태그들을 배워보자! (슈퍼스타들)
title = 책 표지와 같이 사용하는 태그
meta charset = utf-8 -> utf-8로 읽어라
html/head/body 순 링크달기 = a href target = "_blank" title = ""
